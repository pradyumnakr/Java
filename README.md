# Java Learning Repository

Welcome to my Java learning and collection repository! This repository contains a comprehensive collection of Java projects and exercises covering a wide range of topics, from foundational programming concepts to advanced enterprise frameworks like Spring.

## 📁 Repository Structure

The projects are organized into the following main categories:

### 1. Fundamentals & Core Java
- **`Basics/`**: Contains introductory examples covering variables, loops, conditionals, methods, and basic syntax.
- **`OOP/`**: Demonstrates Object-Oriented Programming principles including classes, inheritance, composition, and more.
- **`Advanced/`**: Covers more complex Java features such as Multithreading and Exception Handling.

### 2. Computer Science Concepts
- **`Algorithms/`**: Implementations of classic computer science algorithms, including:
  - Dijkstra's Algorithm
  - Floyd's Algorithm
  - Kruskal's & Prim's Algorithms
  - Sorting (Merge, Quicksort)
  - Traveling Salesperson Problem (TSP) & Knapsack problems
- **`DataStructures/`**: Implementations and usage of fundamental data structures like Stacks.

### 3. Spring Framework Applications
This section includes various projects demonstrating the Spring Framework ecosystem:
- **`SpringDemo/`**: A foundational Spring Framework project demonstrating core dependency injection concepts.
- **`SpringJDBC/`**: An application showcasing database connectivity and interactions using Spring JDBC.
- **`SpringMVC/`**: A Model-View-Controller web application demonstrating how to build web interfaces with Spring.
- **`SpringBoot/`**: A modern Spring Boot application highlighting rapid application development and auto-configuration.

## 🚀 Getting Started

To run these projects locally, ensure you have the Java Development Kit (JDK) installed on your machine.
For the Spring projects, you may need Maven or Gradle installed, though wrappers (`mvnw`, `gradlew`) are provided in the respective directories for convenience.

```bash
# Clone this repository
git clone https://github.com/pradyumnakr/Java.git
```

Navigate into each directory to explore and run the specific subprojects.
